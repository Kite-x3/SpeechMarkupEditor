## О проекте

Кроссплатформенное приложение на Avalonia UI, позволяющее аудиологам:
- Загружать дихотические аудиозаписи
- Визуализировать waveform для левого/правого каналов
- Размечать временные интервалы слов с ручной корректировкой
- Экспортировать результаты в JSON

## Репозитории

Вся активная разработка происходит в данном Git-репозитории https://github.com/Kite-x3/SpeechMarkupEditor, расположенном во внутреннем хранилище GitLab
